# walap

* This is a wallpaper app for your smartphone
* Wallpapers are taken from [api](https://unsplash.com/developers)

---
Technology stack:
* Kotlin
* Mvvm
* RxJava3
* Dagger2

Additional libraries
* Coil
* Pager3
* Room
* Retrofit
